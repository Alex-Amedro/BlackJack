# Projet Blackjack WebApp

Ce projet a pour but d'implémenter un jeu en ligne et multijoueurs de Blackjack, en mettant en pratique une architecture MVC avec SpringBoot. Les joueurs pourront s'inviter, chatter en jeu, et garder leur classement/historique de victoires.
